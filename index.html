Install OpenCV on Mac OSX

Assume OpenCV is downloaded into OPENCVROOT 
$ cd OPENCVROOT
$ mkdir release
$ cd release
$ cmake -G "Unix Makefiles" -D CMAKE_BUILD_TYPE=Release ..
$ make -j8
$ sudo make install
And repeat for Debug build with -D CMAKE_DEBUG_POSTFIX=d
